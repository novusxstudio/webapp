# Spell Icons

This directory should contain the following spell icon images:

## Required Files:
- `lightningstrike.png` - Lightning bolt icon (64x64px recommended)
- `healingcircle.png` - Healing/medical cross icon (64x64px recommended)

## Specifications:
- Format: PNG with transparency
- Size: 64x64 pixels (or larger, will be scaled)
- Style: Should match the game's visual aesthetic
- Background: Transparent

## Usage:
These icons are displayed as overlays on the game board when spells are cast,
appearing for 2 seconds before the turn automatically ends.
